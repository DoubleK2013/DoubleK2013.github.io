level2 dom event

在level2 dom event 中，会发生事件冒泡的有：

  1. 页面类： load、unload、resize
  2. 鼠标类： mouseenter、mouseleave
  3. 系统类： error、abort
  4. 焦点类： focus、blur

关于新东西

身处前端圈子，对于新的技术，往往目不暇接。不经意间也学习了很多，现在回过头看看，却留下了朝秦暮楚的口实。在过去，每有‘新’的技术，没见过的东西总是想着也要掌握，常常也忘乎其文档、代码，奈何却是浅尝辄止。即使动手完成了一两个小demo，也只是照猫画虎，看起来那么个样子，但内里却无甚新意，而且，一段时间之后再拿起也只能从头开始，殆矣。
不过，唯一的好处也就只在有了个博闻的名头，唯缺了强识，看的多只是知道的多而不是懂的多，憾事也。倒是跟外行人侃大山能充当一二门面了，止增笑耳。

故而此处的缘由不过也是一种慰藉罢了。
